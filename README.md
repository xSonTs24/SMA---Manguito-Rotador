# RotaCare — versión monolítica funcional con vision.js original

Esta versión mantiene `vision.js` igual al archivo original entregado y ajusta el flujo desde `app.js` e `index.html`.

## Cómo ejecutar

1. Abre la carpeta del proyecto en VS Code.
2. Ejecuta con Live Server o un servidor local. No abras `index.html` con doble clic (`file://`).
3. Al iniciar el primer ejercicio, el navegador pedirá permiso de cámara. Una vez autorizado, el navegador no debería volver a mostrar el permiso durante la sesión.

## Flujo

- Inicio
- Saber más
- Indicaciones
- Preparación del ejercicio
- Brazo derecho con contador completo
- Cambio de brazo
- Brazo izquierdo con contador completo
- Pantalla de ejercicio completado con contador automático
- Rutina completada

## Notas

- La carpeta `imagenes Rotacare` debe permanecer al mismo nivel que `index.html`.
- La carpeta `music` es opcional. Si no está, la detección y la UI siguen funcionando, pero no sonarán los audios de guía o música de fondo.
- `vision.js` no fue modificado para esta entrega.
